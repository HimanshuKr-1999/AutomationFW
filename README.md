# Selenium Automation Framework

A Selenium Automation Framework built using:

- Java
- Selenium WebDriver
- Cucumber BDD
- TestNG
- Maven
- WebDriverManager

## Project Structure

```
src
 ├── main
 │   └── java
 └── test
     ├── java
     └── resources
```

## Features

- Page Object Model (POM)
- Cucumber BDD
- TestNG Runner
- Cross-browser support
- Config Reader
- Utility Classes
- Extent Reports
- Log4j2 Logging
- Screenshot Capture
- Parallel Execution
